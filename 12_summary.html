단원 마무리

1. HTML과 CSS 활용

웹 게임은 HTML과 CSS, 자바스크립트가 어우러져 제작됨 웹 게임 뿐만 아니라 웹 사이트나 웹 애플리케이션 모두 마찬가지 HTML이나 CSS가 자바스크립트의 역할을 분담할 수 있으면 좋음 특히 애니메이션이나 화면 요소들의 배치는 자바스크립트보다는 HTML과 CSS에 맡기는게 더 좋을 때가 많음

2. 이벤트 루프 분석할 때

이벤트가 많이 발생하는 경우에 프로그램 전체에서 발생하는 이벤트를 모두 분석하면 매우 복잡함 이럴 때는 관련 있는 이벤트만 분석해도 됨 어떠한 이벤트를 분석하는데 영향을 미치지 않는 다른 이벤트가 있다면 해당 이벤트는 이벤트 루프 분석에서 제외해도 됨

3. alert 함수 사용 시 주의점

alert 함수는 현재 진행되는 화면 변경 사항이나 애니메이션을 즉시 멈추고 알림창을 띄우므로 알림창이 뜰 때 마지막 화면 변경 사항이나 애니메이션이 적용되지 않는 경우가 많음 이럴 때는 setTimeout과 함께 호출해서 마지막 화면 변경사항이나 애니메이션이 적용될 시간을 주는 것이 좋음
