단원 마무리

1. setInterval

setTimeout과 비슷한 효과 but setTimeout은 한 번만 실행됨 setInterval은 지정한 시간마다 주기적으로 지정한 함수를 실행

2. clearInterval, clearTimeout

setInterval과 setTimeout 함수는 각각 clearInterval과 clearTimeout 함수로 취소 가능 clearTimeout은 setTimeout에 지정한 함수가 아직 실행되지 않았을 때만 취소 가능

3. 배열.includes

||을 사용한 코드는 배열의 includes 메서드로 반복을 줄일 수 있음

4. removeEventListener

addEventListener로 연결한 함수를 removeEventListener로 제거 가능 단, 연결할 때의 함수와 제거할 때의 함수가 같아야 함

이미지 스프라이트

여러 개의 그림을 하나로 합쳐서 서버에 한 번만 요청하게 하는 것 이미지가 하나로 합쳐져 있으므로 CSS와 자바스크립트로 적절히 잘라서 화면에 표시해야 함
