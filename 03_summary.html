1. 무작위로 숫자 뽑기

무작위로 숫자 뽑을 때 Math.random 메서드를 사용 뽑은 값은 정수가 아니므로 Math.floor나 Math.ceil 같은 메서드를 사용해 정수로 변환 Math.floor: 내림 Math.ceil: 올림 Math.round: 반올림 여러 개의 값을 모아 둘 때 사용: 배열, 객체 배열: 단순한 값들의 나열 객체: 값에 이름이 붙음

3. indexOf와 includes

indexOf와 includes는 배열이나 문자열에 원하는 값이 들어 있는지 찾는 메서드 indexOf는 원하는 값이 들어있다면 해당 인덱스를 알려주고, 들어 있지 않다면 -1을 반환 includes는 true/false를 반환

4. forEach와 map

forEach는 반복문 효과를 내는 배열의 메서드 인수로 함수를 넣고, 이 함수가 각각의 배열 요소들에 순서대로 적용되는 구조 map도 반복문 역할을 하지만, 반환값이 있다는 점에서 forEach와 다름 map: 기존 배열의 요소를 일대일로 다른 값으로 바꿈 + 새로운 배열을 만듦(기존 배열 값이 바뀌는 것X)

5. document.createElement, document.createTextNode

각각 태그와 텍스트를 만드는 메서드 다른 태그에 append나 appendChild 하기 전 까지는 화면에 보이지 않음

6. appendChild와 append

document.createElement, document.createTextNode로 만든 태그나 텍스트를 선택한 태그의 자식 태그로 넣음 appendChild로는 하나만 넣을 수 있고, append를 사용하면 여러 개를 동시에 넣을 수 있음 append로 텍스트를 추가할 때는 document.createTextNode 대신 문자열을 바로 넣어도 됨

배열에서 반복문 역할을 하는 메서드: forEach

answer 배열에 있는 요소들을 forEach 메서드로 순회 forEach: 인수로 함수를 받고, 배열의 요소 하나하나에 인수로 받은 함수를 각각 적용함 이때 요소 순서대로 함수를 적용하므로 반복문의 역할을 함 forEach의 인수 배열의 메서드 Array(9)는 길이가 9인 배열을 만듦 fill(): 배열의 요소로 undefined를 채워 넣는 메서드 map: 요소들을 일대일로 짝지어서 다른 값으로 변환하는 메서드 / 함수를 인수로 받음 / return 값에 따라 새로운 요소를 반환 required: 입력값이 필수 type="text" : 입력값이 문자열로 인식 minlength, maxlength: 입력값의 최소 길이 / 최대 길이 pattern: 정규표현식(regular expression)으로 입력값을 검사 (^(?!.*(.).*\1)\d{4}$ - 중복되지 않은 4자리 숫자)
